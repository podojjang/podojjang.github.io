# 개요
    React 작성 방법 알아보기


## 1. 작성 방법
    const Home = () => {
        return (
            <div>
                Hello Next!
            </div>
        );
    }

    export default Home;

## 2. 참고
    react는 import React from 'react'를 적어야 한다
    next는 pages 폴더 내의 js는 컴포넌트로 인식한다
    pages 폴더내에 js가 있다면 이 구문이 필요 없다
